# Tic Tac Toe

A simple browser-based Tic Tac Toe game built with vanilla HTML, CSS, and JavaScript.

## How to Play

1. Open `index.html` in your browser
2. Players take turns clicking cells — Player X goes first
3. First to get 3 in a row (horizontally, vertically, or diagonally) wins
4. Click **Restart Game** to reset the board

## Files

- `index.html` — game layout
- `style.css` — styles
- `game.js` — game logic
