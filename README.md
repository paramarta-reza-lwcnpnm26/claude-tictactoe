# Tic Tac Toe

[![CI](https://github.com/paramarta-reza-lwcnpnm26/claude-tictactoe/actions/workflows/claude.yml/badge.svg)](https://github.com/paramarta-reza-lwcnpnm26/claude-tictactoe/actions)

A simple browser-based Tic Tac Toe game built with vanilla HTML, CSS, and JavaScript — no dependencies, no build step.

## Live Demo

Open `index.html` directly in your browser — no server needed.

## How to Play

1. Open `index.html` in your browser
2. Players take turns clicking cells — Player X goes first
3. First to get 3 in a row (horizontally, vertically, or diagonally) wins
4. Click **Restart Game** to reset the board

## Tech Stack

- HTML — game layout
- CSS — styling and animations
- JavaScript — game logic (win detection, turn management)

## Project Structure

```
├── index.html   # Game layout
├── style.css    # Styles
└── game.js      # Game logic
```
